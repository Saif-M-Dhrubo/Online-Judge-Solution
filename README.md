# Online Judges Solution
Solution for Algorithm based problems in various online judges ([**UVa**](https://uva.onlinejudge.org/), [**LightOJ**](http://www.lightoj.com/index.php) and [**Codeforces**](http://codeforces.com/)).

**Topics Covered :**
- Graph Theory
- Number Theory
- Dynamic Programming
- Greedy Technique
- String Matching
- Divide and Conquer
